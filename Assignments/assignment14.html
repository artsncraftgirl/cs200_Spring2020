Assignment #14 - Language of Choice: Hello World

Purpose
Explore programming in another language
Instructions
  1. Select a programming language other that HTML/CSS/JavaScript.
  2. Find documentation on that language and setup the development environment on your computer.
  3. Write the traditional "Hello World program in this language".
  4. Get a copy of your code in that language. Get a screen capture of the output in whatever form. Write a paragraph or more with your impressions of the new language. Put all these elements in a page. Test to make sure the buttons work and resolve any bugs.
  5. Submit the link to the page in Canvas.
